# my_blog
게시글을 작성하여 나만의 블로그를 만들 수 있는  API  
[블로그 프로젝트 정보](https://dev-rara.notion.site/Blog-Project-6c0667ca7016470b8599a5599d8a92b4)
<br>

## 🛠️Tech Stack
* Spring Boot
* Spring Data JPA
* Spring Validation
* JWT
* AWS RDS(MySQL)
* AWS EC2
* Lombok  <br>

**5. 선택한 게시글 조회 API**
