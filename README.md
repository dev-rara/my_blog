# my_blog
게시글을 작성하여 나만의 블로그를 만들 수 있는 Rest API  
[블로그 프로젝트 정보](https://dev-rara.notion.site/Blog-Project-6c0667ca7016470b8599a5599d8a92b4)
<br>

## 🛠️Tech Stack
* Spring Boot
* Spring Data JPA
* Lombok
* H2 Database  <br>

**1. 회원가입 API** 

**2. 로그인 API**

**4. 게시글 작성 API**

**5. 선택한 게시글 조회 API**

**6. 선택한 게시글 수정 API**

**7. 선택한 게시글 삭제 API**
  
**8. 예외처리**
